# ai-adaptive-onboarding-engine
Datasets used:
- O*NET Technology Skills database — used to validate and expand 
  skill catalog coverage across 923 occupations
- Kaggle Resume Dataset (snehaanbhawal, 2484 resumes) — used to 
  test extraction accuracy across 24 job categories
- Kaggle Jobs Dataset (kshitizregmi) — used to validate skill 
  overlap detection across real-world job descriptions
